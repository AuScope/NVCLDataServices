NVCL Data Services : Get Best TSA Results

Error

getBestTSAResults service will generate a table of best TSA results in csv or json format. It requires a holeidentifier or a datasetid as the mandatory URL parameter. The holeidentifier can be obtained from gml:id of either gsml:Borehole or gsmlp:BoreholeView WFS output (e.g. gsml.borehole.WTB5, take the WTB5 only).

Usage: getBestTSAResults.html?parameter1=value

ParameterDescriptionRequiredExample values
holeidentifiersame as borehole id which can be obtained through the GeoServer WFS ouputyesWTB5
datasetidalternatively you can use the dataset's own ID to look up a datasetno6dd70215-fe38-457c-be42-3b165fd98c7
outputformatoutput format. options are csv and json. default is csvnojson

A standard query : getBestTSAResults.html?holeidentifier=WTB5
