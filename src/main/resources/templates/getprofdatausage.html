NVCL Data Services : Get prof Data

Error

This service will get prof data in a binary chunk. It requires a proflogid that can be obtained through the getDatasetCollection service. You can optionally provide a start and end sample number to restrict the range of data returned. The data will be returned as 4 byte floating point numbers. You will need to use the prof log's 'wavelengths' string array to determine how many floats to expect per sample.

ParameterDescriptionRequiredExample values
proflogidobtained by looking at the prof logs returned by calling the getDatasetCollection serviceyes3923f5f9-08f3-415f-8503-7578ab1d784
startsamplenothe sample number of the first profilometer data point to be returnedno, Default value=00
endsamplenothe sample number of the last profilometer data point to be returnedno, Default value=99999910
outputformatbinary or jsonnojson

Example Queries:

A stadard query for prof data: getprofdata.html?speclogid=3923f5f9-08f3-415f-8503-7578ab1d784&startsampleno=0&endsampleno=10
