NVCL Data Services : Get Image Section Service

Error

Given a valid dataset id and section number, this service builds the corresponding section's image and returns it as a JPEG.

Usage: getImageSection.html?parameter1=value&parameter2=value

ParameterDescriptionRequiredExample values
datasetidthe dataset's ID yes6dd70215-fe38-457c-be42-3b165fd98c7
sectionnosection number of the section you want to retrieve(0 based index)yes0
horizontalturn the vertical downhole image horizontal as it would appear in a traynoyes

Example Queries:

A stadard query for retrieve the first section of imagery from a dataset: getImageSection.html?datasetid=6dd70215-fe38-457c-be42-3b165fd98c7&sectionno=0

the same query as above but rotated horizontally : getImageSection.html?datasetid=6dd70215-fe38-457c-be42-3b165fd98c7&sectionno=0&horizontal=yes
