NVCL Data Services : Get Dataset Depth Range

Error

the getDatasetDepthRange service will return the dataset depth range for a provided dataset id. It requires a datasetid as the mandatory URL parameter. The datasetid can be obtained through the getDatasetCollection service.

ParameterDescriptionRequiredExample values
datasetidobtained through calling the getDatasetCollection serviceyes0bf74ac7-1642-481c-b0fc-689d2d3f904
