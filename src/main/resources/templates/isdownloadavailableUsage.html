NVCL Data Services : isdownloadavailable

Error

isdownloadavailable service will check if the dataset identified by its name or ID has a TSG file available for download and if so get its download url. It requires either a datasetid or datasetname.

Usage: isdownloadavailable.html?parameter1=value

ParameterDescriptionRequiredExample values
datasetidUUID of the datasetno6dd70215-fe38-457c-be42-3b165fd98c7
datasetnameAlternatively you can provide the dataset's namenoWTB5
outputformatchange output format from xmlnojson

Example Queries:

A standard query : isdownloadavailable.html?datasetid=6dd70215-fe38-457c-be42-3b165fd98c7
